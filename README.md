### Version 0.1 (27/03/2017)
* First map
* First map's music
* Scroll
* Screen Edges
* Character (Yuan-nang)
* Character Movement (Yuan-nang) :
     * ↑ ➲ Move player up
     * ↓ ➲ Move player down
     * ← ➲ Move player left
     * → ➲ Move player right
     * ↑ + → ➲ 
     * ↑ + ← ➲ 
     * ↓ + → ➲
